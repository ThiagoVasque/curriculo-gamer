# Currículo Gamer

O Currículo Gamer é um gerenciador de biblioteca de jogos pessoal, focado em transformar a experiência de catalogar games em um portfólio visual. O projeto foi desenvolvido para oferecer uma interface imersiva e organizada para jogadores que desejam documentar sua jornada.

## Sobre o projeto

A ideia central é permitir que o usuário gerencie seu progresso de forma detalhada. O sistema se integra à base de dados da IGDB para buscar informações oficiais, garantindo que a biblioteca tenha dados precisos e capas em alta resolução.

### Principais recursos:

* **Busca integrada:** Localização de títulos através da API da IGDB (Twitch).
* **Sistema de avaliação:** Notas de 1 a 10 com suporte a meias estrelas para maior precisão.
* **Organização de status:** Divisão entre jogos finalizados, favoritos e títulos que ainda estão na fila (backlog).
* **Interface Dark:** Design focado em legibilidade e estética moderna.

## Tecnologias utilizadas

Este projeto utiliza o ecossistema Laravel para o backend e Tailwind CSS para a construção da interface visual.

* **Framework:** Laravel 11
* **Frontend:** Tailwind CSS / Blade Components
* **Banco de Dados:** MySQL
* **Autenticação:** Laravel Breeze (Customizado)
* **API Externa:** IGDB API
